# homebrew-tap

Homebrew tap for [LeetCode Tracker](https://github.com/rustamPy/leetcode-tracker) — a macOS menu bar app.

## Install

```bash
brew tap rustamPy/tap
brew install --cask leetcode-tracker
```

## Update

```bash
brew upgrade --cask leetcode-tracker
```

## Uninstall

```bash
brew uninstall --cask leetcode-tracker
```
